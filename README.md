# ALOS — Second Mind

A local-first founder cognitive operating system.

## Working features
- Natural-language capture
- Automatic classification into decisions, risks, tasks, patterns, goals, and memories
- Interactive Obsidian-style knowledge graph
- Search
- Highest-ROI recommendation
- Feedback tracking
- JSON backup
- Mobile-first interface

Data is stored in browser localStorage for this MVP. No external AI platform or API key is required.
